# Git & GitHub Fundamentals

This module covers the essential workflow for version control and remote collaboration.

## 🛠️ Concepts & Commands Mastered

### Configuration & Setup
- Installation and local authentication.
- Git identity configuration (`user.name` and `user.email`).

### Basic Workflow
- `git init`: Initializing a new repository from scratch.
- `git add .`: Managing the **Staging Area**.
- `git commit -m ""`: Creating snapshots of the project.
- `git push / git pull`: Synchronizing local and remote repositories.

### Branching & Collaboration
- **Branches:** Understanding isolated development environments.
- **Merge:** Integrating changes from different branches.

### Profile & Portfolio
- Creating professional `README.md` files.
- Project publication and profile customization.
