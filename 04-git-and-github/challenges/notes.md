# Git & GitHub - Key Takeaways

### My Biggest Learning
Understanding the **Staging Area** and the **Commit History** was a game-changer. I learned that Git is not just about saving files, but about creating a timeline of a project's evolution.

### Key Concepts Mastered:
- **Version Control:** How to keep track of changes without losing previous work.
- **Remote Collaboration:** Using `push` and `pull` to keep local and remote repositories in sync.
- **Troubleshooting:** Learning how to fix terminal states (like the `dquote>` issue) and cleaning up system junk with `.gitignore`.

*This module transformed my workflow from a "copy-paste" method to a professional version control standard.*
