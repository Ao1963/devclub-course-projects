# 01 - CSS Introduction

In this folder, I explored the transition from internal styles to external stylesheets. I separated my progress into 'basic' and 'enhanced' exercises to demonstrate my understanding of CSS architecture.

### What's inside:
*   **Internal Approach:** Practice with styles directly inside the HTML document.
*   **External Standard:** Implementing professional stylesheets using the `link` tag.
*   **Separation of Concerns:** My first steps into keeping structure (HTML) and presentation (CSS) organized and separate.
