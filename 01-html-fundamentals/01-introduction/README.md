### 🧠 Core Concepts: What is HTML?

*   **Definition:** HTML stands for **Hypertext Markup Language**.
*   **Purpose:** It is the standard language used to create and structure content on the web. It acts as the "skeleton" of a website.
*   **Communication:** It is a way to communicate with the browser, telling it how to display content by "marking" or "labeling" elements.
*   **Not a Programming Language:** It's important to note that HTML is a **markup language**, not a programming language, as it doesn't handle logic or calculations.
*   **Structure & Semantics:** HTML is responsible for defining elements such as buttons, headings, paragraphs, and links, providing the essential foundation for any web project.
